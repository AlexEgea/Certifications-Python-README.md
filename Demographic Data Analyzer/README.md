# 🌍 Demographic Data Analyzer

Proyecto desarrollado como parte de la certificación **Python for Data Analysis** de freeCodeCamp.

---

## 📌 Descripción

Este proyecto analiza datos demográficos reales del censo de 1994 utilizando **Pandas**.

Se realizan distintos cálculos estadísticos y análisis sobre:

* Edad
* Nivel educativo
* Salario
* País de origen
* Ocupación

---

## 📊 Análisis realizados

El programa responde a preguntas como:

* Número de personas por raza
* Edad media de los hombres
* Porcentaje de personas con estudios superiores
* Relación entre educación y salario
* Horas mínimas de trabajo
* País con mayor porcentaje de ingresos altos
* Ocupación más común en India (>50K)

## 🧠 Funcionalidades clave

* Limpieza de datos (eliminación de espacios)

* Cálculo de porcentajes

## 📈 Resultados destacados

* Edad media hombres: **39.4**
* % con estudios superiores que ganan >50K: **46.5%**
* País con mayor porcentaje de ingresos altos: **Iran**
* Ocupación más común en India: **Prof-specialty**
