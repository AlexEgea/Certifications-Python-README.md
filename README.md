# 🐍 Python Certification Projects – Data Analysis

Repositorio dedicado a los proyectos realizados durante la certificación de **Python for Data Analysis** de freeCodeCamp.

## 📌 Descripción

Este repositorio recoge distintos proyectos enfocados al análisis de datos utilizando Python y sus principales librerías.
El objetivo es demostrar habilidades prácticas en:

* Manipulación de datos
* Análisis estadístico
* Uso de NumPy y Pandas
* Limpieza y transformación de datos
* Resolución de problemas reales

## 🚀 Proyectos incluidos

### 1. 📊 Mean-Variance-Standard Deviation Calculator

Calcula métricas estadísticas básicas (media, varianza, desviación estándar, etc.) a partir de una matriz 3x3 utilizando **NumPy**.

🔧 Tecnologías:

* Python
* NumPy

---

### 2. 🌍 Demographic Data Analyzer

Análisis de datos demográficos basado en el dataset del censo de 1994 utilizando **Pandas**.

Incluye:

* Análisis por raza
* Estadísticas salariales
* Comparación por nivel educativo
* Análisis por país
* Identificación de patrones laborales

🔧 Tecnologías:

* Python
* Pandas

## ⚙️ Instalación

Clonar el repositorio:

```bash

cd python
```

Instalar dependencias:

```bash
pip install -r requirements.txt
```
