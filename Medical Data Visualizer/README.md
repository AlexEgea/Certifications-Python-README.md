# 🩺 Medical Data Visualizer

Proyecto de análisis de datos médicos utilizando Python, Pandas, Matplotlib y Seaborn.

## 📊 Descripción

Este proyecto analiza datos médicos de pacientes para identificar patrones relacionados con enfermedades cardiovasculares.

Se realizan dos visualizaciones principales:

* 📊 **Cat Plot**: Comparación de variables como colesterol, glucosa, actividad, etc.
* 🔥 **Heatmap**: Matriz de correlación entre variables

## ⚙️ Tecnologías

* Python
* Pandas
* Matplotlib
* Seaborn
